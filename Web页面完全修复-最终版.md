## 🔧 如果仍有问题

### 问题A: 示例1成功，示例3失败

**原因**: 直接ZPL转换有问题

**检查**: 
```bash
# 测试转换功能
python -c "from src.utils.zpl_chinese_converter import detect_and_convert_zpl; zpl='^^XA^FO20,10^A0N,25,25^FD测试^FS^XZ'; result, converted=detect_and_convert_zpl(zpl); print(f'原始: {len(zpl)}'); print(f'转换后: {len(result)}'); print(result)"
```

应该看到转换后的ZPL保持^XA和^XZ。

### 问题B: 所有中文都是空白

**原因**: Pillow或字体问题

**解决**:
```bash
pip install Pillow

# 检查字体
dir C:\Windows\Fonts\msyh.*
```

### 问题C: 打印机只闪灯

**检查debug文件**:
```bash
# 查看最新的debug文件
Get-ChildItem data\debug_web_*.zpl | Sort-Object LastWriteTime | Select-Object -Last 1 | Get-Content
```

确认:
- 以^XA开头
- 以^XZ结尾
- 包含^GFA图像数据
