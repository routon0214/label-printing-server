# 纸张匹配问题解决方案

## 问题现象

打印机报错："没有匹配到对应的纸张"

## 诊断结果

当前打印机配置：
- **ZT411 标签打印机**: 104.0 x 50.8 mm (4.09 x 2.00 英寸) - 标签纸
- **KONICA 文档打印机**: A4 (210 x 297 mm) - A4纸

## 快速解决方案

### 方案1：调整打印机纸张设置（最简单）⭐ 推荐

#### KONICA 文档打印机

1. **打开打印机设置**
   ```
   Windows 设置 → 设备 → 打印机和扫描仪
   ```

2. **选择打印机**
   - 找到 "KONICA MINOLTA Universal PCL"
   - 点击"管理" → "打印首选项"

3. **设置纸张**
   - 纸张大小：**A4** 或 **Letter**（根据实际纸张选择）
   - 纸张来源：**自动选择** 或指定纸盒
   - 缩放：**适应纸张大小** 或 **无缩放**

4. **高级设置**（可选）
   - 找到"纸张/质量"选项卡
   - 启用"自动选择纸张"
   - 或启用"忽略纸张大小不匹配"

5. **点击确定保存**

#### ZT411 标签打印机

1. **打开打印机设置**
   - 找到 "ZDesigner ZT411-300dpi ZPL"
   - 点击"管理" → "打印首选项"

2. **设置标签纸**
   - 纸张大小：选择实际标签纸尺寸
   - 常见尺寸：
     - 4x6 英寸（102x152mm）
     - 4x2 英寸（102x51mm） ← 当前
     - 自定义尺寸

3. **打印方式**
   - 打印方法：热敏或热转印
   - 打印速度：中等
   - 打印浓度：调整至合适

修改 PDF 打印时的行为，让 SumatraPDF 自动适应纸张：

#### 临时测试方法

在 `core/pdf_printer.py` 中，SumatraPDF 命令已经使用 `-silent` 参数，这应该会自动适应纸张。

**如果仍然报错，尝试以下操作**：

   - 设置为"继续"或"忽略"

2. **使用默认打印机测试**
   ```bash

   # 然后运行测试
   python tests/test_auto_print.py
   ```

### 方案3：修改文档大小以匹配纸张

如果您的文档不是 A4 大小，可以：

1. **创建 A4 大小的测试文档**
   ```python
   from reportlab.pdfgen import canvas
   from reportlab.lib.pagesizes import A4
   
   c = canvas.Canvas("test_a4.pdf", pagesize=A4)
   c.drawString(100, 750, "Test A4 Document")
   c.save()
   ```

2. **或使用 Letter 大小**（美国标准）
   ```python
   from reportlab.lib.pagesizes import letter
   c = canvas.Canvas("test_letter.pdf", pagesize=letter)
   ```

### 方案4：使用 Windows 命令行打印

创建一个批处理脚本直接打印：

```batch
@echo off
REM print_test.bat
REM 使用默认打印机打印

rundll32 printui.dll,PrintUIEntry /y /n "KONICA MINOLTA Universal PCL"
SumatraPDF.exe -print-to-default -silent "test.pdf"
```

## 针对不同场景的建议

### 场景1：打印 PDF 文档到 KONICA

**问题**：PDF 文档大小与 A4 纸不匹配

**解决**：
1. 在打印首选项中启用"缩放以适应纸张"
2. 或创建 A4 大小的 PDF

### 场景2：打印标签到 ZT411

**问题**：ZPL 标签尺寸与打印机设置不匹配

**解决**：
1. 确保 ZPL 代码中的尺寸与打印机纸张设置一致
2. 或在打印机设置中修改纸张为实际标签尺寸

### 场景3：文本文件打印

**问题**：文本文件被当作 PDF 处理，但格式不对

**解决**：
1. 文本文件会被 SumatraPDF 识别为文本
2. 确保打印机设置为 A4 或 Letter
3. 或使用 notepad 直接打印（仅文本）

