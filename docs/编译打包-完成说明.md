## 🔧 已解决的问题

### 编码问题
**问题**: Windows 控制台 GBK 编码无法显示 UTF-8 字符
**解决**: 
1. 在 `build.py` 中添加编码修复代码
2. 替换特殊 Unicode 字符为 ASCII 兼容字符 ([OK], [ERROR] 等)

```python
# Windows 控制台编码修复
if sys.platform == 'win32':
    import io
    if sys.stdout.encoding != 'utf-8':
        sys.stdout = io.TextIOWrapper(sys.stdout.buffer, encoding='utf-8', errors='replace')
    if sys.stderr.encoding != 'utf-8':
        sys.stderr = io.TextIOWrapper(sys.stderr.buffer, encoding='utf-8', errors='replace')
```

## 🐛 常见问题

### Q: 打包失败怎么办？
A: 
1. 确保安装了 PyInstaller：`pip install pyinstaller`
2. 清理旧文件：删除 `build` 和 `dist` 目录
3. 重新运行打包脚本

### Q: 找不到模块怎么办？
A: 在 `app.spec` 的 `hiddenimports` 中添加缺失的模块

### Q: 可执行文件太大？
A: 
1. 使用虚拟环境减少依赖
2. 在 `app.spec` 中排除不需要的模块

### Q: 杀毒软件报警？
A: 
1. 添加到白名单
2. 考虑代码签名
