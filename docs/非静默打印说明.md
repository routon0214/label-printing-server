# 非静默打印说明

## 变更说明

系统已从**静默打印**模式改为**显示打印对话框**模式。

### 为什么做此改变？

**静默打印的问题**：
- ❌ 对纸张大小要求严格
- ❌ 无法手动调整打印设置
- ❌ 纸张不匹配时直接失败
- ❌ 用户无法控制打印过程

**显示对话框的优势**：
- ✅ 用户可以手动选择打印机
- ✅ 用户可以调整纸张大小
- ✅ 用户可以设置打印份数、颜色等
- ✅ 避免纸张不匹配问题
- ✅ 更灵活、更可控

## 使用示例

### 通过 MQTT 发送打印任务

```python
import paho.mqtt.client as mqtt
import json
import base64

client = mqtt.Client()
client.connect("10.100.10.121", 1883)

# 读取 PDF 文件
with open("document.pdf", "rb") as f:
    pdf_base64 = base64.b64encode(f.read()).decode('utf-8')

# 发送打印任务
message = {
    "print_type": "pdf",
    "pdf_data": pdf_base64
}

client.publish("zebra/print", json.dumps(message))
client.disconnect()
```

## 命令行参数变更

### SumatraPDF

**之前（静默）**：
```bash
SumatraPDF.exe -print-to "打印机名称" -silent file.pdf
```

**现在（对话框）**：
```bash
SumatraPDF.exe -print-dialog file.pdf
```

### Adobe Reader

**之前（静默）**：
```bash
AcroRd32.exe /t file.pdf "打印机名称"
```

**现在（对话框）**：
```bash
AcroRd32.exe /p file.pdf
```

### Notepad

**保持不变**（本来就有对话框）：
```bash
notepad.exe /p file.txt
```

## 优缺点对比

| 特性 | 静默打印 | 对话框打印 |
|-----|---------|-----------|
| 自动化程度 | 高 | 低 |
| 用户交互 | 无 | 需要 |
| 灵活性 | 低 | 高 |

| 纸张适配 | 严格 | 灵活 |
| 适用场景 | 自动化环境 | 办公环境 |

## 适用场景

### 推荐使用（对话框模式）

✅ **办公环境**
- 用户需要确认打印内容
- 需要调整打印设置
- 打印机可能更换

✅ **混合纸张**
- 不同文档使用不同纸张
- 需要手动选择纸盒
- 需要调整缩放

✅ **测试环境**
- 需要验证打印效果
- 频繁调整设置
- 实验性打印

### 不推荐使用（对话框模式）

❌ **完全自动化环境**
- 无人值守打印
- 批量打印任务
- 生产环境

❌ **高频打印**
- 每天数百张
- 需要快速响应
- 标准化流程

## 如果需要静默打印

如果您确实需要静默打印（不显示对话框），可以：

### 选项1：恢复静默模式

修改 `core/pdf_printer.py` 中的命令参数：

```python
# SumatraPDF - 改回静默
cmd = [
    sumatra_found,
    '-print-to', printer_name,
    '-silent',  # 添加这个参数
    pdf_file
]
```

### 选项2：使用 win32print RAW 模式

对于支持 RAW 打印的文档（文本、ZPL）：
- 已经是静默的
- 直接发送到打印队列
- 不显示对话框

### 选项3：混合模式

- PDF/DOC：显示对话框（需要用户确认）
- ZPL 标签：静默打印（自动化）
- TXT 文本：根据需要选择

## 常见问题

### Q: 为什么要取消静默打印？

**A**: 静默打印对纸张大小要求非常严格，经常因为纸张不匹配而失败。显示对话框后，用户可以手动调整设置，避免打印失败。

### Q: 每次都要手动点击"打印"吗？

**A**: 是的。但这样可以：
- 确认打印内容正确
- 调整打印设置（纸张、份数等）
- 选择合适的打印机
- 避免浪费纸张

### Q: 标签打印也要手动确认吗？

**A**: 不需要。ZPL 标签打印仍然是自动的，不显示对话框。

### Q: 可以部分文档静默、部分显示对话框吗？

**A**: 可以。通过修改代码，为不同的打印类型设置不同的行为。

## 技术细节

### SumatraPDF 参数

| 参数 | 说明 |
|-----|------|
| `-print-to <打印机>` | 静默打印到指定打印机 |
| `-print-to-default` | 静默打印到默认打印机 |
| `-print-dialog` | 显示打印对话框 |
| `-silent` | 静默模式（不显示窗口）|

### 当前使用

- ✅ `-print-dialog` - 显示对话框
- ❌ ~~`-silent`~~ - 已移除
- ❌ ~~`-print-to`~~ - 已移除
