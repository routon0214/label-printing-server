## 技术要点

### 斑马打印机 (ZPL)

**中文处理方式**:
```python
# 自动检测并转换中文
from src.utils.zpl_chinese_converter import detect_and_convert_zpl

zpl_code, was_converted = detect_and_convert_zpl(original_zpl)
# 如果包含中文，was_converted = True
# 中文会被转换为 ^GFA 图像命令
```

**工作流程**:
1. 检测ZPL代码中是否包含中文字符
2. 如果有中文，使用Pillow库渲染为黑白图像
3. 将图像转换为十六进制格式
4. 生成 `^GFA` 图像嵌入命令
5. 替换原有的 `^FD...^FS` 文本命令

**优点**:
- 支持所有中文字符
- 显示效果美观

**缺点**:
- 需要Pillow库和中文字体
- 生成的ZPL代码较长

### 得力打印机 (ESC/POS)

**中文处理方式**:
```python
# 使用GB2312编码
data = {
    'raw_text': '测试打印\n中文内容\n',
    'encoding': 'gb2312'  # 关键！
}
printer.print_receipt(data)
```

**工作流程**:
1. 发送初始化命令: ESC @ (0x1B 0x40)
2. 设置字符集: ESC t 14 (0x1B 0x74 0x0E) = GB2312
3. 使用GB2312编码发送文本数据

**优点**:
- 无需图像转换
- 打印速度快
- 代码简单

**缺点**:
- 只支持GB2312字符集中的字符
- 依赖打印机内置中文字体

### 常见问题

#### 斑马打印机中文显示方块
**原因**: 缺少中文字体或Pillow库
**解决**:
```bash
pip install Pillow

# Windows: 确认字体
dir C:\Windows\Fonts\msyh.*

# Linux: 安装字体
sudo apt-get install fonts-wqy-zenhei
```

#### 得力打印机中文乱码
**原因**: 未使用GB2312编码或未设置字符集
**解决**: 
- 确保 `encoding='gb2312'`
- 已自动添加 ESC t 14 命令

#### Web界面打印中文失败
**原因**: Web应用未自动转换中文
**解决**: 已在 `web_app.py` 中集成自动转换功能
