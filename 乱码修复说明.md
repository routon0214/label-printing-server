# 乱码问题完全解决方案

## 问题类型和解决方法

### 1. Windows控制台日志显示乱码 ✅ 已修复

**问题原因**: 
- 代码中使用了Unicode特殊字符（如 ✓ ✗ ⚠ → 等）
- Windows PowerShell/CMD默认编码不是UTF-8

**解决方案**:
1. ✅ **已完成**: 运行 `python scripts/fix_unicode_chars.py` 将所有特殊字符替换为ASCII兼容字符
   - ✓ → [OK]
   - ✗ → [ERROR]  
   - ⚠ → [WARNING]
   - → → ->
   - 已修复31个文件

2. **查看日志**: 双击运行 `查看日志.bat` (自动使用UTF-8编码)

3. **启动服务**: 使用 `start_utf8.bat` (自动设置UTF-8编码)

### 2. 实际打印输出乱码

#### A. ZPL标签打印乱码

**原因**: 
- 缺少中文字体
- Pillow库未安装

**检查**:
```bash
python scripts/diagnose_encoding.py
```

**解决**:
```bash
# 安装Pillow
pip install Pillow

# Windows: 确认字体存在
dir C:\Windows\Fonts\msyh.*
dir C:\Windows\Fonts\simhei.*

# Linux: 安装字体
sudo apt-get install fonts-wqy-zenhei fonts-wqy-microhei
```

#### B. ESC/POS热敏打印机乱码

**原因**: 
- ✅ 已修复: 代码已使用GB2312编码
- ✅ 已修复: 已添加字符集设置命令 (ESC t 14)

**测试**:
```bash
python tests/test_escpos_chinese.py
```

**MQTT消息格式**:
```json
{
  "print_type": "receipt",
  "raw_text": "测试文本\n中文打印",
  "encoding": "gb2312"
}
```

### 3. Web界面显示乱码

**检查**: 
1. 浏览器开发者工具 -> Network -> 查看响应头
2. 确认 `Content-Type: text/html; charset=utf-8`

**文件检查**:
- `templates/index.html` 应包含: `<meta charset="UTF-8">`
- 文件本身应为UTF-8编码（无BOM）

### 4. 上传文件乱码

**原因**: 上传的文件编码不是UTF-8

**解决**:
```bash
# Windows: 使用记事本
# 1. 打开文件
# 2. 另存为 -> 编码选择 "UTF-8" (不要选BOM)

# Linux: 使用iconv
iconv -f GBK -t UTF-8 input.txt > output_utf8.txt
```

## 常见问题

### Q1: PowerShell中仍然看到乱码？
A: 使用 `查看日志.bat` 或设置PowerShell编码:
```powershell
[Console]::OutputEncoding = [System.Text.Encoding]::UTF8
chcp 65001
```

### Q2: 打印出来的内容是乱码？
A: 
- ZPL: 检查字体和Pillow库
- ESC/POS: 已自动使用GB2312编码
- PDF: 确认PDF文件本身没有乱码

### Q3: 日志文件太大？
A: 日志自动轮转，最大10MB，保留5个文件

### Q4: 需要重新构建exe吗？
A: 是的，如果要打包为exe，需要重新构建:
```bash
python scripts/build.py
```

## 技术说明

### 编码选择原则

1. **ZPL标签打印**:
   - 文件: UTF-8
   - 中文: 转换为图像 (不直接打印中文字符)
   - 依赖: Pillow + 中文字体

2. **ESC/POS热敏打印**:
   - 编码: GB2312/GBK
   - 命令: ESC t 14 (设置简体中文字符集)
   - 原因: 打印机硬件只支持GB2312

3. **PDF打印**:
   - 文件: UTF-8
   - 处理: PDF内嵌字体

4. **日志文件**:
   - 编码: UTF-8 (with BOM is not recommended)
   - 查看: 需要支持UTF-8的工具
