# 快速开始指南

## 🎯 5分钟快速启动

### 第一步：安装依赖

```bash
pip install -r requirements.txt
```

### 第二步：配置打印机

编辑 `config/printer_config.json`：

```json
{
  "mqtt": {
    "host": "127.0.0.1",
    "port": 1883,
    "topic": "zebra/print"
  },
  "printer": {
    "name": "ZT411"
  }
}
```

💡 **提示**：打印机名称支持模糊匹配，"ZT411" 会自动匹配 "ZDesigner ZT411-300dpi"

### 第三步：启动服务

**方式一：使用启动脚本（推荐）**

Windows:
```batch
start.bat
```

Linux:
```bash
chmod +x start.sh
./start.sh
```

**方式二：直接运行**

```bash
python app.py
```

### 第四步：发送测试消息

打开新的终端窗口：

```bash
cd tests
python test_mqtt_send.py
```

## 📡 发送打印任务

### Python示例

```python
import paho.mqtt.client as mqtt
import json

# 连接MQTT
client = mqtt.Client()
client.connect("127.0.0.1", 1883, 60)

# 准备标签数据
label_data = {
    "title": "产品标签",
    "fields": [
        {"label": "产品名称", "value": "精密电子元件", "font_size": 28},
        {"label": "序列号", "value": "SN20251015001", "font_size": 22}
    ],
    "barcode": "SN20251015001",
    "qrcode": "SN20251015001"
}

# 发送消息
client.publish("zebra/print", json.dumps(label_data))
client.disconnect()
```

### 命令行示例（使用mosquitto_pub）

```bash
mosquitto_pub -h 127.0.0.1 -t zebra/print -m '{
  "title": "测试标签",
  "fields": [
    {"label": "测试项", "value": "测试值", "font_size": 25}
  ]
}'
```

## 🔧 常见配置

### 配置网络打印机

```json
{
  "printer": {
    "ip": "192.168.1.100",
    "port": 9100
  }
}
```

### 配置MQTT认证

```json
{
  "mqtt": {
    "host": "mqtt.example.com",
    "port": 1883,
    "username": "user",
    "password": "pass"
  }
}
```

### Linux CUPS打印

```json
{
  "printer": {
    "name": "Zebra-ZT411"
  }
}
```

### Linux USB直连

```json
{
  "printer": {
    "device": "/dev/usb/lp0"
  }
}
```

## 🐛 问题排查

### 找不到打印机

程序会自动列出所有可用打印机：

```
发现 2 台打印机:
  1. Microsoft Print to PDF
  2. ZDesigner ZT411-300dpi
```

在配置文件中使用正确的名称或关键词。

### MQTT连接失败

1. 检查MQTT服务器是否运行
2. 检查防火墙设置
3. 验证主机名和端口

### 中文显示乱码

```bash
# 安装Pillow
pip install Pillow

# Linux安装中文字体
sudo apt-get install fonts-wqy-zenhei
```
