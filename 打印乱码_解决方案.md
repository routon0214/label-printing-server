## 🔬 技术原理说明

### ESC/POS打印机中文编码原理

根据参考案例 `D:\workspaces\py-scripts\print.py` 的成功经验：

```python
# 关键代码
init_sequence = b'\x1B\x40'      # ESC @ - 初始化打印机
charset_command = b'\x1B\x74\x0E'  # ESC t 14 - 设置简体中文GB2312
data_to_send = init_sequence + charset_command + raw_data.encode('gb2312')
```

**为什么必须使用GB2312而不是UTF-8？**

1. **历史原因**: 大多数中文ESC/POS打印机（如得力、佳博）基于早期设计，内置GB2312字符集
2. **字符集映射**: ESC t n 命令设置打印机的字符代码表，n=14 对应 GB2312
3. **直接映射**: GB2312编码的字节可以直接映射到打印机内置字体，无需转换
4. **UTF-8不兼容**: 打印机硬件不支持UTF-8的多字节序列

**修复前后对比**:

```python
# 修复前（错误）
commands.extend(text.encode('utf-8'))  # ❌ 打印机无法识别

# 修复后（正确）
commands.extend(b'\x1B\x74\x0E')      # ✓ 设置GB2312字符集
commands.extend(text.encode('gb2312'))  # ✓ 使用正确编码
```
