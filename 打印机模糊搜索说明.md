# 打印机模糊搜索功能说明

## 功能概述

程序支持**智能模糊搜索**功能，无需输入完整的打印机名称，即可自动匹配到正确的打印机。

## 使用场景

### 1. 配置文件中使用

**传统方式**（需要完整名称）：
```json
{
  "printer": {
    "name": "ZDesigner ZT411-300dpi ZPL"
  }
}
```

**模糊搜索方式**（简短关键词）：
```json
{
  "printer": {
    "name": "ZT411"
  }
}
```
程序会自动匹配到 `ZDesigner ZT411-300dpi ZPL`

### 2. 自动检测

即使不配置打印机名称，程序也会自动使用模糊搜索查找最合适的打印机。

## 匹配规则

模糊搜索按照以下优先级匹配：

### 1. 完全匹配（100分）
```
输入: "ZDesigner ZT411-300dpi"
匹配: "ZDesigner ZT411-300dpi" ✓
```

### 2. 开头匹配（90分）
```
输入: "ZDesigner"
匹配: "ZDesigner ZT411-300dpi" ✓
```

### 3. 包含匹配（60-80分）
```
输入: "ZT411"
匹配: "ZDesigner ZT411-300dpi ZPL" ✓
匹配: "Zebra ZT411" ✓
```

位置越靠前，分数越高。

### 4. 分词匹配（30-60分）
```
输入: "zebra 411"
匹配: "Zebra ZT411" ✓
匹配: "ZDesigner ZT411-300dpi" ✓
```

检查是否包含所有关键词。

### 5. 字符顺序匹配（10-40分）
```
输入: "z411"
匹配: "Zebra ZT411" ✓
```

检查字符是否按顺序出现。

## 配置优先级

1. **网络打印** (`ip`) - 跳过名称匹配
2. **模糊搜索** (`name`) - 使用模糊匹配
3. **设备路径** (`device`) - 直接使用
4. **自动检测** (`null`) - 使用预设关键词模糊搜索

## 常见问题

### Q: 为什么匹配到错误的打印机？

A: 关键词太通用。例如 `printer` 可能匹配到 `Microsoft Print to PDF`。

**解决方案**：使用更具体的关键词，如型号 `ZT411`。

### Q: 为什么找不到打印机？

A: 关键词太特殊或打印机名称不包含该关键词。

**解决方案**：
1. 查看程序输出的打印机列表
2. 使用更通用的关键词
3. 使用完整名称

### Q: 如何知道打印机的完整名称？

A: 程序启动时会显示所有打印机：

```
发现 3 台打印机:
  1. Microsoft Print to PDF
  2. ZDesigner ZT411-300dpi ZPL
  3. Fax
```

或者：
- **Windows**: 控制面板 → 设备和打印机
- **Linux**: `lpstat -p` 或 http://localhost:631

### Q: 最低匹配分数是多少？

A: 最低30分。低于30分认为不匹配，会提示失败。

### Q: 能否完全禁用模糊搜索？

A: 可以，使用完整的打印机名称即可（分数100，优先匹配）。
