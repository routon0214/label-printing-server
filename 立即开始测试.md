## 🔧 故障排查

### 中文是空白

**原因**: Pillow未安装或中文字体缺失

**解决**:
```bash
pip install Pillow

# 检查字体
dir C:\Windows\Fonts\msyh.*
```

### 中文是乱码

**得力打印机**: 应该使用GB2312编码
→ 检查控制台是否显示"使用GB2312编码"

### 完全没打印

**检查**:
1. 打印机是否开机、有纸
2. 控制台是否显示"打印成功"
3. 打印机队列是否卡住

**诊断**:
```bash
python tests/test_zd888_chinese.py
# 选择 5 - 检查打印机状态
```

## 💡 技术说明

### Web应用已集成的功能

**1. 文件上传自动处理**:
- 检测ZPL代码中的中文
- 自动调用 `detect_and_convert_zpl()`
- 将中文转换为图像

**2. JSON打印自动处理**:
- 使用 `ZPLGenerator` 生成ZPL
- 内部自动调用 `text_to_image_zpl()`
- 中文自动转换为 `^GFA` 图像命令

**3. ESC/POS自动处理**:
- 自动使用GB2312编码
- 自动添加字符集设置命令 (ESC t 14)

### 关键代码位置

```python
# web_app.py 第512-514行
zpl_code, was_converted = detect_and_convert_zpl(zpl_code)
if was_converted:
    print("  [OK] ZPL中文已自动转换为图像")

# web_app.py 第635-637行

    print("  [OK] JSON中的ZPL代码，中文已自动转换为图像")
```
