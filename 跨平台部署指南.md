# 斑马打印机MQTT服务 - 跨平台部署指南

支持 Windows、Linux (AMD64/ARM) 平台

## 系统架构

```
MQTT消息 → zebra_mqtt_printer.py → 斑马打印机
                                     ↓
                     ┌────────────────┼────────────────┐
                     │                │                │
                 Windows            Linux           网络打印
                (win32print)       (CUPS/USB)     (所有平台)
```

## 平台依赖

### Windows

```bash
pip install paho-mqtt Pillow pywin32
```

### Linux (AMD64/ARM)

```bash
# 基础依赖
pip install paho-mqtt Pillow

# CUPS打印（推荐）
sudo apt-get install cups python3-cups
pip install pycups

# 或者使用USB设备直接打印（无需额外依赖）
```

## 二、Linux部署

### 2.1 CUPS打印（推荐）

#### 安装CUPS

```bash
# Ubuntu/Debian
sudo apt-get update
sudo apt-get install cups python3-cups

# CentOS/RHEL
sudo yum install cups python3-cups

# Arch Linux
sudo pacman -S cups python-pycups
```

#### 配置打印机

```bash
# 启动CUPS
sudo systemctl start cups
sudo systemctl enable cups

# 添加打印机（Web界面）
# 访问 http://localhost:631
# Administration → Add Printer

# 或使用命令行
lpadmin -p Zebra-ZT411 -v usb://Zebra/ZT411 -E
```

#### 配置文件

```json
{
  "printer": {
    "name": "Zebra-ZT411"
  }
}
```

#### 查看CUPS打印机

```bash
lpstat -p -d
```

### 2.2 USB设备直接打印

#### 查找USB设备

```bash
ls -l /dev/usb/lp*
# 或
ls -l /dev/bus/usb/
```

#### 设置权限

```bash
sudo chmod 666 /dev/usb/lp0
# 或永久设置
sudo usermod -a -G lp $USER
```

#### 配置文件

```json
{
  "printer": {
    "device": "/dev/usb/lp0"
  }
}
```

### 2.3 网络打印（通用）

```json
{
  "printer": {
    "ip": "192.168.1.100",
    "port": 9100
  }
}
```

### 2.4 安装依赖

```bash
pip3 install paho-mqtt Pillow
# 如果使用CUPS
pip3 install pycups
```

### 2.5 启动服务

```bash
python3 zebra_mqtt_printer.py
```

### 2.6 systemd服务

创建服务文件：`/etc/systemd/system/zebra-printer.service`

```ini
[Unit]
Description=Zebra MQTT Printer Service
After=network.target

[Service]
Type=simple
User=your_user
WorkingDirectory=/opt/zebra-printer
ExecStart=/usr/bin/python3 /opt/zebra-printer/zebra_mqtt_printer.py
Restart=always
RestartSec=10

[Install]
WantedBy=multi-user.target
```

启动服务：

```bash
sudo systemctl daemon-reload
sudo systemctl start zebra-printer
sudo systemctl enable zebra-printer
sudo systemctl status zebra-printer
```

查看日志：

```bash
sudo journalctl -u zebra-printer -f
```

## 五、配置文件说明

### 完整配置

```json
{
  "mqtt": {
    "host": "mqtt.example.com",
    "port": 1883,
    "topic": "zebra/print",
    "username": "printer",
    "password": "secret"
  },
  "printer": {
    "name": null,
    "ip": null,
    "port": 9100,
    "device": null
  }
}
```

### 打印机配置优先级

1. **网络打印** (`ip`) - 最高优先级，所有平台通用
2. **打印机名称** (`name`) - Windows/CUPS
3. **设备路径** (`device`) - Linux USB
4. **自动检测** - 如果都为null

### 配置示例

#### Windows USB
```json
{"printer": {"name": "ZDesigner ZT411-300dpi"}}
```

#### Linux CUPS
```json
{"printer": {"name": "Zebra-ZT411"}}
```

#### Linux USB
```json
{"printer": {"device": "/dev/usb/lp0"}}
```

#### 网络打印（通用）
```json
{"printer": {"ip": "192.168.1.100", "port": 9100}}
```

## 七、故障排查

### Windows

**问题：找不到打印机**
```bash
# 检查打印机名称
控制面板 → 设备和打印机 → 右键属性
```

**问题：缺少pywin32**
```bash
pip install pywin32
```

### Linux

**问题：CUPS打印机列表为空**
```bash
# 检查CUPS状态
sudo systemctl status cups

# 添加打印机
http://localhost:631
```

**问题：USB设备权限不足**
```bash
sudo chmod 666 /dev/usb/lp0
sudo usermod -a -G lp $USER
# 注销重新登录
```

**问题：找不到中文字体**
```bash
# 安装字体
sudo apt-get install fonts-wqy-zenhei

# 验证
fc-list :lang=zh
```

### ARM

**问题：Pillow安装失败**
```bash
# 安装编译依赖
sudo apt-get install libjpeg-dev zlib1g-dev
pip3 install Pillow
```

## 九、安全建议

1. **MQTT认证**：使用用户名密码
2. **TLS加密**：MQTT over SSL
3. **防火墙**：限制访问IP
4. **权限控制**：最小权限原则

## 十、Docker部署

```dockerfile
FROM python:3.9-slim

# 安装依赖
RUN apt-get update && \
    apt-get install -y fonts-wqy-zenhei && \
    rm -rf /var/lib/apt/lists/*

RUN pip install paho-mqtt Pillow

# 复制程序
COPY zebra_mqtt_printer.py /app/
COPY printer_config.json /app/

WORKDIR /app

CMD ["python", "zebra_mqtt_printer.py"]
```
