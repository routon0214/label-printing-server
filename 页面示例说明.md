### 3. 直接ZPL代码（含中文）⭐ 重要

**用途**: 验证Web自动转换功能

**数据**:
```json
{
  "print_type": "label",
  "zpl_code": "^XA^FO50,50^A0N,40,40^FD你好世界^FS^FO50,120^A0N,30,30^FDHello^FS^XZ"
}
```

**关键**: 这个ZPL代码直接包含中文"你好世界"

**Web应用处理**:
1. 检测到`zpl_code`字段
2. 调用`detect_and_convert_zpl()`
3. 自动将"你好世界"转换为^GFA图像命令
4. 保持^XA和^XZ结构完整

**控制台输出**:

## 关键区别说明

### JSON生成 vs 直接ZPL

**方式A: JSON生成**（示例1、2、5）
```json
{
  "print_type": "label",
  "title": "测试",
  "fields": [...]
}
```

处理流程:
```
JSON数据 → ZPLGenerator.generate_label_zpl()
         → 内部自动调用 text_to_image_zpl()
         → 生成包含^GFA的完整ZPL
         → 发送打印
```

**方式B: 直接ZPL**（示例3、4）
```json
{
  "print_type": "label",
  "zpl_code": "^XA^FO50,50^A0N,40,40^FD测试^FS^XZ"
}
```

处理流程:
```
直接ZPL → detect_and_convert_zpl()
        → 检测中文 → 替换^FD中文^FS为^GFA图像
        → 保持^XA和^XZ结构
        → 发送打印
```

## 重要修复

### 修复前的问题

**旧的TSPL示例**（错误）:
```json
{
  "print_type": "label",
  "zpl_code": "SIZE 60 mm, 40 mm\nTEXT ..."
}
```
- 这是TSPL格式，不是ZPL
- 只适用于TSC打印机
- ZT411无法识别

### 修复后

所有示例都是**标准ZPL格式**：
- ✅ 以^XA开头
- ✅ 以^XZ结尾
- ✅ 使用ZPL命令（^FO, ^A, ^FD等）
- ✅ 适用于Zebra ZT411打印机
